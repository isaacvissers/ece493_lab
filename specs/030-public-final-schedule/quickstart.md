# Quickstart: Public Final Schedule Announcement

## Prerequisites
- Node.js and npm installed

## Run
1. Start the CMS app
2. Publish a schedule for public release
3. Visit the public schedule page

## Verify
- Public schedule page loads after publish
- Schedule shows day/time/room/session and paper titles/authors/abstracts
- “Schedule not available yet” appears before release
- Friendly error appears on render failure
- “Last updated” timestamp appears after updates
